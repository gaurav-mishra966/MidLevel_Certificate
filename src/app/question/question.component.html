Problem Statement

In this application, we want to filter a list of GitHub repositories based on name and programming language. The app uses components/repository.component.ts and its existing template to display each repository in the list. When the user changes the "Name" or "Programming Language", the page must refresh and filter the list of repositories. Both filters must work separately and must also work together.
